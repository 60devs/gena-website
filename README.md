## Gena &emdash; A Static Site Generator built with Gulp and Handlebars

### Install Gena

```sh
git clone https://orkon@bitbucket.org/60devs/gena.git
cd gena
npm install
npm link gena
```

### Dependencies for this site instance

Install jspm

```sh
npm install jspm -g
```

or update it

```sh
npm update jspm -g
```
### Get a copy of this website

```sh

jspm dl-loader
jspm install

gena s

```
